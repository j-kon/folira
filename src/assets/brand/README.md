# Folira Brand Assets Catalog

This directory contains the primary vector SVG source assets for Folira.

* `folira-symbol.svg`: Core geometric brand mark.
* `folira-wordmark.svg`: Folira typography wordmark.
* `folira-logo-horizontal.svg`: Horizontal mark + wordmark lockup.
* `folira-logo-stacked.svg`: Stacked mark + wordmark lockup.
* `folira-logo-light.svg`: Light theme logo.
* `folira-logo-dark.svg`: Dark theme logo.
* `folira-monochrome.svg`: Monochrome logo.

Production PNG assets (favicons, PWA icons, Open Graph image) are generated from `folira-symbol.svg` using `scripts/generate-brand-assets.ts`.
